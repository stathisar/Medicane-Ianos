Current repository includes various scripts, performance metrics and sample checks regarding the research published in OAJAST of OPAST publishers with title:

Exploiting multiple social media sources and multiple modalities for severe weather management: The case study of the Medicane Ianos.

All the sample checks performed in the research paper, exist. They had been submitted to the journal along with the manuscript. There are some sample checks online, regarding photo classification, without though the actual image files. And some model metrics despite the modality, as those are referred in the paper. Some other data though like actual tweet or instagram caption samples are not online. In case you are interested to view the referred or other material that cannot be found in the repository feel free to contact. There are privacy matters regarding social media data, thus it was decided not to have sensitive info online.
